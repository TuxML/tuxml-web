# API RETRIEVE JSON
  
  Allows to fetch compilations information with the use of multiple arguments.

## URL:

	api/v1/resources/compilations

## Method:
  
	`GET`
  
## URL Parameters:

	**Optional:**

	`cid=[integer]`
	`compiled_kernel_version=[alphanumeric]`
	`gcc_version=[alphanumeric]`
	`compiled=["true", "false"]`
	`display=["architecture", "cid", "compilation_date", "compilation_time", "compiled_kernel_size", "compiled_kernel_version", "compressed_compiled_kernel_size", "config_file,", "cpu_brand_name", "cpu_max_frequency", "dependencies", "gcc_version", "libc_version", "linux_distribution", "linux_distribution_version", "mechanical_disk", "number_cpu_core", "number_cpu_core_used", "ram_size", "stderr_log_file", "stdout_log_file", "system_kernel", "system_kernel_version", "tuxml_version", "user_output_file"]`
	`limit=[integer]`

	**Syntax:**
	add a ? after compilations, followed by one of the URL parameters. 
	Seperate each parameter with a '&'.
	Separate each display paramter with a comma ','.	

## Success Response:
  
  * **Code:** 200 <br />
    **Content:** `cid=[integer]`
 
## Error Response:

  * **Code:** 400  <br />
    **Content:** `ERROR 400 : [argument] is not formatted as expected`

  * **Code:** 404 NOT FOUND <br />
    **Content:** `ERROR 404 : No compilation exists with a [argument] of [argument_value]`

## Sample Call:

	api/v1/resources/compilations?compiled_kernel_version=5.8&limit=50&compiled=false&display=cid,compilation_date
  
## Notes:

	The default limit is 100.
	
# API RETRIEVE JSON:
	
	Allows the user to upload a compilation's information to the database. Needs the use of an authorization token.

## URL:
	
	api/v1/uploadResults

## METHOD

	`POST`
	
##Data Parameters:

	`--header 'Authorization: Bearer [authorization_token]'`

## Success Response:
  
  * **Code:** 201 <br />
    **Content:** à compléter
	
## Error Response:

  * **Code:** 400  <br />
    **Content:** `ERROR 400 : The json isn't complete`

  * **Code:** 409 NOT FOUND <br />
    **Content:** à compléter
	
  * **Code:** 500 NOT FOUND <br />
    **Content:** `ERROR 500: The upload has failed but keep calm, it's our fault`
